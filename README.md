## HoloNet 控制台 · AI 手势全息网络面板

HoloNet 控制台是一个基于 **MediaPipe 手势识别 + React + Three.js** 的全息可视化控制台，
通过浏览器摄像头捕获手势，在三维地球与 HUD 面板上实时联动，模拟「AI 驱动的网络监控 / 运维中台」场景。

核心特性：

- **手势交互**：通过摄像头识别单手 / 双手姿态，完成旋转、缩放、拖拽等交互操作。
- **三维地球可视化**：使用 `@react-three/fiber` 和 `three.js` 渲染全息地球，用于承载「网络节点 / 区域」信息。
- **HUD 信息面板**：在屏幕上叠加系统状态、帧率、识别状态等信息，营造未来感操作界面。
- **纯前端实现**：所有逻辑在浏览器完成，适合部署到静态托管或边缘计算平台（如阿里云 ESA Pages）。

## 本地运行

**环境依赖：** Node.js (建议 18+)

1. 安装依赖：

   ```bash
   npm install
   ```

2. 启动开发环境：

   ```bash
   npm run dev
   ```

3. 按终端提示在浏览器中访问本地地址（通常为 `http://localhost:5173`）。

> 提示：首次进入页面需要允许浏览器访问摄像头，才能体验完整的手势交互效果。

## 使用说明

1. 打开页面后，允许浏览器访问摄像头。
2. 在启动界面点击中间的电源按钮，进入 HoloNet 控制台主界面。
3. 手势操作：
   - **左手**：在画面左侧移动手掌，向前/向后推，可以拉近或拉远全息地球，模拟「缩放视角」。
   - **右手捏合（拇指 + 食指）**：
     - 捏住时抓取地球，移动手掌可以拖拽地球在画面中的位置；
     - 松开时地球会缓慢回到默认锚点。
   - **特殊手势**：竖起中指会触发一次「安全演示模式」，HUD 上会展示“锁定异常节点 → 模拟流量清洗 → 节点隔离”的过程，仅为可视化演示，不进行任何真实操作。
4. 右侧 HUD 会展示：系统时间、帧率、识别状态以及当前聚焦区域等信息，后续也可以接入真实网络指标。

## 技术实现与 ESA 适配

- **前端技术栈**：
  - React + TypeScript 负责状态管理与 HUD / 控制台 UI；
  - `@react-three/fiber` + `three.js` 渲染全息地球与空间光效；
  - MediaPipe `HandLandmarker` 进行实时手部关键点检测。

- **部署形态**：
  - 使用 Vite 构建为纯静态资源，适合托管在阿里云 ESA Pages：
    - 安装命令：`npm install`
    - 构建命令：`npm run build`
    - 静态资源目录：`dist`

- **可扩展的边缘应用形态**：
  - 控制台当形式展示手势 + 三维可视化能力，后续可以：
    - 通过 ESA 边缘函数对接真实网络诊断后端（如延迟、丢包、带宽等指标），将数据映射到地球表面的不同区域；
    - 借助 ESA 的 KV / 缓存存储最近会话与告警记录，在 HUD 中展示「近 5 分钟告警」等信息；
    - 在边缘侧完成基础数据聚合，让前端只需拉取轻量化的 JSON 状态即可完成渲染。

这样，HoloNet 控制台既可以作为一个炫酷的人机交互，也可以自然演进为基于 ESA 的边缘网络监控面板，兼顾创意性、应用价值和技术深度。
